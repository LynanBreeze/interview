## 什么是闭包

一个可以操作外部其他函数定义的局部变量的函数，相当于一个变量作用域，作用域可以向外操作，但不可向内操作。

## 逃不过的继承方式

### 原型链继承

问题：
1. 包含引用类型值的原型属性会被所有实例共享，这也是为什么要在构造函数中，而不是原型对象中定义属性的原因。
2. 没有办法在不影响所有对象实例的情况下，给超类型的构造函数传递参数
